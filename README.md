# 📊 Job Supply Analytics Dashboard

## 🚀 Project Overview

The **Job Supply Analytics Dashboard** is a data analytics project developed as part of **PlaceMux Phase 2 – Data Analyst Task 2**. The goal of this project is to validate job posting records, analyze hiring activity, and create an interactive dashboard that provides insights into job supply trends across the marketplace.

The solution leverages **SQL** for data validation, cleaning, and analysis, while **Power BI** is used to build an interactive dashboard for monitoring recruitment activity and business performance.

---

## 🎯 Business Problem

Organizations need visibility into job posting activity to understand:

- Hiring demand
- Marketplace growth
- Recruitment trends
- Skill requirements across roles

This dashboard answers key business questions such as:

- How many jobs are being posted?
- Which companies are posting the most jobs?
- How are job postings trending over time?
- What skill thresholds are commonly required?
- What percentage of jobs are active versus closed?

---

## 📌 Objectives

- Validate job posting records
- Track overall job supply
- Analyze company-wise hiring activity
- Monitor job posting trends
- Identify skill threshold patterns
- Build an interactive Power BI dashboard for decision-making

---

## 🗂️ Dataset Information

The dataset contains job posting information including:

| Column | Description |
|----------|-------------|
| Job ID | Unique job identifier |
| Company Name | Hiring organization |
| Job Title | Position name |
| Job Category | Job classification |
| Skill Threshold | Required skill score |
| Posted Date | Date job was posted |
| Job Status | Active or Closed |
| Other Attributes | Additional job-related information |
